## 最强领域

- 端到端编译流水线（C 源码 → 可执行文件）
- 完整的 C 语言前端（类型系统、控制流、预处理器）
- 双目标代码生成（Windows x64 + Linux x64）
- 内置 PE/COFF 链接器
- 彩色诊断输出
- 自动化回归测试（218 个用例）

## 最弱领域

1. **优化器深度** — CSE 有 bug 被禁用，缺少 SSA/CFG，寄存器分配未优化
2. **错误恢复** — 解析器遇到第一个错误即停止，无拼写建议
3. **标准合规** — 不完整的算术转换、有限的未定义行为检测
4. **ELF 支持** — 依赖 WSL gcc，非内置链接器

## 下一步计划

1. **功能完善** — 修复 CSE、完善 PE 导出表
2. **架构改进** — 拆分 CodeGenerator、引入 CFG、Visitor 模式统一 AST 遍历
3. **优化** — 寄存器分配、优化级别控制、SSA 中间表示
4. **工具链** — DWARF 调试信息、编译器性能优化
5. **链接器** — PE 异常处理表、增量链接、更好的错误诊断
